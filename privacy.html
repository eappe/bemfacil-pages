Política de Privacidade – Bem Fácil

O aplicativo Bem Fácil coleta apenas as informações necessárias para criação e gerenciamento da conta do usuário, incluindo nome e endereço de email.

1. Dados coletados

  • Nome
  • Email

Esses dados são usados exclusivamente para autenticação e identificação do usuário dentro do aplicativo.

2. Base tecnológica

O app utiliza Firebase Authentication e Firestore para armazenamento seguro de dados.

3. Finalidade do uso dos dados

Os dados são utilizados para:

  • Login e autenticação
  • Gerenciamento da conta
  • Suporte ao usuário, quando necessário

4. Compartilhamento

Os dados não são compartilhados com terceiros para fins comerciais. Apenas provedores de serviço tecnológicos (como Firebase) processam dados conforme necessário para operação do app.

5. Direitos do usuário

O usuário pode solicitar a exclusão de sua conta e dados entrando em contato pelo email: contato@eappe.com.br.
